# RG351P-Wi-Fi-Adapter
RG351P adapter board for Wi-Fi modules

#### Compatible modules:
  - **HLK-7601U2**
  - **BL-R7601MU2**
  - **RTL8188EUS**
  - **RL-UM12BS-8188EUS**
